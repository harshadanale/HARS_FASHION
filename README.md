HARS — Women's Fashion SS'25

A luxury women's clothing & lifestyle brand based in Koregaon Park, Pune, India.

This is the official brand website — a fully self-contained, single-file web experience built with pure HTML, CSS, and Vanilla JavaScript.


🌸 About HARS
HARS is a contemporary women's fashion label focused on effortless silhouettes, bold statements, and pieces designed for the modern woman. From streetwear-inspired wide-leg edits to luxury power blazers and garden-romantic dresses — HARS curates fashion that wears confidence.
